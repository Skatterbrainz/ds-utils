# Show-ExplorerMenuBar

## SYNOPSIS
Enable Display of Menu Bar in Windows Explorer

## SYNTAX

```
Show-ExplorerMenuBar [-Enable] <Boolean> [-AllUsers] [<CommonParameters>]
```
