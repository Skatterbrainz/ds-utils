# Show-FileExtensions

## SYNOPSIS
Enable display of file extensions in Windows Explorer

## SYNTAX

```
Show-FileExtensions [-Enable] <Boolean> [-RestartShell] [<CommonParameters>]
```

## DESCRIPTION
Enable display of file extensions in Windows Explorer

## EXAMPLES

### EXAMPLE 1
```
Show-FileExtensions -Enable $True -RestartShell
```
