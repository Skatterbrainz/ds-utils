# Set-NugetApiKey

## SYNOPSIS
Set-DsNugetApiKey

## SYNTAX

```
Set-NugetApiKey [-ApiKey] <String> [<CommonParameters>]
```

## DESCRIPTION
Stores NuGet API key in registry

## EXAMPLES

### EXAMPLE 1
```
Set-NugetApiKey -ApiKey '01234567-abcd-4567-defg-987654321aaa'
```

## PARAMETERS

### -ApiKey
NuGet API key (GUID) from repository (e.g.
PowerShell Gallery)
