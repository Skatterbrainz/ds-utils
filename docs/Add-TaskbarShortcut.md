# Add-TaskbarShortcut

## SYNOPSIS
Pin Shortcut to Taskbar

## SYNTAX

```
Add-TaskbarShortcut [-Path] <String> [<CommonParameters>]
```

## DESCRIPTION
Pin Shortcut to Taskbar

## EXAMPLES

### EXAMPLE 1
```
Add-TaskbarShortcut -Target "c:\windows\notepad.exe"
```

## PARAMETERS

### -Path
