# Add-DsTaskbarShortcut

## SYNOPSIS
Pin Shortcut to Taskbar

## SYNTAX

```
Add-DsTaskbarShortcut [-Target] <String> [<CommonParameters>]
```

## DESCRIPTION
Pin Shortcut to Taskbar

## EXAMPLES

### EXAMPLE 1
```
Add-DsTaskbarShortcut -Target "c:\windows\notepad.exe"
```

## PARAMETERS
